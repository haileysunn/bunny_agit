# 기능 명세서

## 1. 지도 기능

### 지도 표시
- Kakao Maps API 활용
- 현재 위치 자동 이동
- 마커로 흡연구역 표시
- 내 위치로 이동 버튼

### 마커 인터랙션
- 마우스 오버: 정보창 표시 (이름, 실내/실외)
- 클릭: 상세 정보 모달 열기

### 검색 기능
- 아지트 이름 검색
- 주소 검색
- 실시간 필터링
- 검색 결과 토스트 표시

---

## 2. 아지트 제보

### 제보 정보
- 장소명 (필수)
- 주소 (필수)
- 위도/경도 (필수, GPS 자동 입력 가능)
- 실내/실외 체크박스

### 검증 시스템
- 100m 이내 제보만 허용
- 50m 이내 동일 장소 판단
- 3명 이상 제보 시 지도에 표시
- 제보 횟수 추적

---

## 3. 리뷰 시스템

### 리뷰 작성
- 청결도 평가 (1-5점)
- 이용 가능 여부 체크
- 코멘트 (선택)

### 리뷰 표시
- 평균 청결도 계산
- 최신순 정렬
- 이용 가능/불가 상태 표시

### 신뢰도 점수
- 최근 7일 리뷰 기반
- 이용가능률 60% + 청결도 40% 가중치
- 색상 코드: 초록(80%+), 노랑(50-80%), 빨강(50% 미만)
- 최근 이용불가 경고 표시

---

## 4. UI/UX 특징

### 다크모드
- 헤더 토글 버튼 (🌙/☀️)
- Tailwind dark 클래스 적용

### 키보드 단축키
- ESC: 모달 닫기

### 로딩 상태
- 데이터 로딩 시 스피너 표시
- 제보/리뷰 제출 중 버튼 비활성화

### 토스트 알림
- 성공/오류/정보 메시지
- 3초 후 자동 사라짐
- 슬라이드 인 애니메이션
